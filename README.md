## 開発経緯
私自身受験生のときに、塾でたくさんの講義を受けてきたが、勉強の仕方が確立せず成績が伸び悩んだ経験があった。
どうやって勉強すればよいか悩んだ時に、”自分にあった学習方法を相談できる先生”がいれば精神的にも時間的にも楽になる。
そこで、「頑張りたいけど、どう頑張ったらいいかわからない！」というジレンマを抱えた学生さんをサポートするアプリを作りたいと思った。

## サービス概要
- 自分の学習目的に合った相談ができる先生を一覧から見つけることができる。
- 先生が自分オリジナルの”コース”を用意することができ、生徒がそのコースに申し込むことができる。
- 先生と生徒がチャットルームでやり取りすることができ、申し込み前のコース内容を相談したり、コース受講中の質問や相談をすることができる。

## 機能概要
- 先生一覧から先生個々のプロフィール閲覧。
- 相談したい内容をピンポイントで条件指定できる検索機能。
- 先生/生徒とのチャット機能。（受講前の相談も可。）
- 生徒から先生への受講申請と先生側での承認機能。
- 生徒から先生への受講の終了申請と先生側での承認機能。
- 受講履歴（生徒ページ）、実績（先生ページ）の閲覧。

## 使用技術
- React.js 
- Next.js 
- tailwindcss
- typescript
